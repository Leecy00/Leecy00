코드의 재사용성 
1. 퀴즈를 랜덤으로 뽑아서 중복을 제거하는 코드는 로또번호 프로그램과 같이 배열의 값을 랜덤으로 뽑고
중복으로 제거할 때 재사용할 수 있다.
2. 빙고 판별 코드는 반복문을 통하여 각각 가로,세로, 대각선을 빙고 판별하였다. 이로써 유사한 게임을 만들어 각
줄의 값이 동일한지 판별할 때 이 코드를 리팩토링하여 재사용할 수 있다.

기대효과
1.각 반복문의 기능을 나누었다. 
랜덤으로 뽑아서 새 배열에 저장하는 기능
문제를 출력하는 기능
정답과 오답을 판별하는 기능
빙고를 판별하는 기능
빙고 여부를 출력하는기능

이로써 코드가 하나의 기능을 맡고있어 버그가 있거나 개선해야 할때 
해당코드만 수정하면 되니 유지보수가 편한 장점을 찾을수 있었다.

2.히든 카드기능을 사용하여 더욱 사용자의 재미를 유발 할 수 있을 것이다.
